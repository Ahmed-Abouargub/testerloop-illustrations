# Testerloop feature illustrations

Self-contained bundle for the Testerloop site's feature-scroll section: three live,
interactive illustrations (test playback, parallel testing, easy integration) that
replace the static feature SVGs on the Webflow page.

`feature-illustrations.js` injects its own CSS and mounts each illustration over the
matching `.feature-scroll-image` slot. Loaded on the Webflow page via jsDelivr.

Generated from source illustrations — do not hand-edit.
